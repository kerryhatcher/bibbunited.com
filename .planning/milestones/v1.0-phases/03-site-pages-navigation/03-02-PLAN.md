**2. DateDisplay** at `src/components/shared/DateDisplay.tsx` (per D-14, DSGN-06):
- Client Component (`'use client'`) to avoid hydration mismatch on relative dates (per pitfall 4 from research)
- Import `formatDistanceToNow`, `format`, `differenceInDays` from `date-fns`
- Export helper function `formatArticleDate(dateString: string): string`:
  - Parse date: `const date = new Date(dateString)`
  - Calculate: `const daysAgo = differenceInDays(new Date(), date)`
  - If `daysAgo < 7`: return `formatDistanceToNow(date, { addSuffix: true })` (e.g., "2 days ago")
  - Else: return `format(date, 'MMMM d, yyyy')` (e.g., "March 15, 2026")
- Export component `DateDisplay({ publishDate, updatedAt, variant = 'full' }: Props)`:
  - Props: `publishDate: string`, `updatedAt?: string`, `variant?: 'full' | 'compact'`
  - Use `useState` and `useEffect` to set formatted dates client-side (render absolute date initially, update to relative in useEffect to prevent hydration mismatch)
  - `variant='full'` (for article pages): Show "Published: March 15, 2026" and if updatedAt differs from publishDate by more than 1 day, also show "Updated: March 20, 2026" â€” both dates use `format(date, 'MMMM d, yyyy')` for full display
  - `variant='compact'` (for cards/lists): Show single formatted date using `formatArticleDate` (relative for < 7 days, absolute otherwise)
  - Render with `suppressHydrationWarning` on the time elements as a safety net
  - Style: `text-text-secondary text-sm`

  ```typescript
  export async function generateMetadata({ params }: Args) {
    const { slug } = await params
    const payload = await getPayload({ config: configPromise })
    const result = await payload.find({
      collection: 'news-posts',
      where: { slug: { equals: slug } },
      limit: 1,
    })
    const post = result.docs[0]
    if (!post) return { title: 'Not Found' }
    return { title: `${post.title} | BIBB United` }
  }
  ```
