<context>
This is a greenfield civic advocacy website for the BIBB community. The site's core mission is to inform community members about local school system issues (budgets, policies, board decisions) and activate them to take action (attend meetings, contact officials). The editorial team is small (2-3 people) and non-technical, so the CMS admin experience is critical.

Key research finding: Payload CMS 3.x is architecturally different from 2.x — it runs embedded inside Next.js, not as a separate Express server. Old tutorials and blog posts referencing 2.x patterns will lead to broken implementations. Lean on official Payload 3.x docs.

The user runs their own K8s cluster with Traefik and Cloudflare tunnels, indicating strong infrastructure experience. Deployment target is a single Docker container.
</context>
