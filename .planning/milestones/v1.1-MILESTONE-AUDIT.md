## Unsatisfied Requirements

### VIS-02: Navigation menu populated with all site sections (PARTIAL)

**Root cause:** `src/seed.ts` line 469 seeds the Contact Officials navigation link with URL `/officials`, but the actual route is `/contact-officials`. Clicking this link from the nav results in a 404.

**Fix:** Change `url: '/officials'` to `url: '/contact-officials'` in seed.ts.

### UX-02: Active nav indicator with aria-current (PARTIAL)

**Root cause:** Same seed URL mismatch. `isActiveLink()` in Header.tsx compares `pathname === href`. When the user is on `/contact-officials` but the href is `/officials`, the comparison fails. The active indicator and `aria-current="page"` never activate for Contact Officials.

**Fix:** Same as VIS-02 — correcting the seed URL resolves both issues.
