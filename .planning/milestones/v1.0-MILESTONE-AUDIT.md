- **Unused `User` icon import** in `contact-officials/page.tsx` — imported from `lucide-react` but never rendered in JSX. Likely intended as fallback avatar for officials without photos. Cosmetic dead code, no functional impact.
- **Homepage `latestNews` render query omits `_status: published` filter** (`page.tsx` lines 47-52) — safe in production because `NewsPosts` collection access control enforces published-only for unauthenticated users. During development with admin session active, drafts may appear in the homepage latest news section. Affects DSGN-03 in dev only.
