# Phase 2 — UI Design Contract

> Visual and interaction contract for the BIBB United brand identity and design system. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (manual Tailwind v4 CSS-first) |
| Preset | not applicable |
| Component library | none (custom components, no Radix/Base UI) |
| Icon library | lucide-react |
| Heading font | Barlow Condensed (700) via next/font/google |
| Body font | Inter (400, 700) via next/font/google |

**Source:** D-06, D-07, D-09 from CONTEXT.md. Font choices within Claude's discretion: Barlow Condensed selected for its protest-poster impact at condensed widths while maintaining legibility; Inter selected for clean readability on long-form civic content.

---

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, icon-to-label gaps |
| md | 16px | Default element padding, form field padding |
| lg | 24px | Card internal padding, component spacing |
| xl | 32px | Section padding on mobile |
| 2xl | 48px | Section padding on tablet+ |
| 3xl | 64px | Major section breaks, page-level vertical rhythm |

Component size constraints (not spacing tokens):
- Touch targets: minimum 44px height (buttons, interactive elements) per WCAG 2.1 AA.
- Hero section vertical padding: minimum 80px. This is a component-level size constraint driven by D-16 ("generous spacing between sections"), not a reusable spacing token.

**Source:** Claude's discretion (Tailwind spacing scale). D-16 from CONTEXT.md ("generous spacing between sections") drives the 80px hero constraint.

## Typography

| Role | Size | Weight | Line Height | Transform | Font |
|------|------|--------|-------------|-----------|------|
| Body / Label | 16px | 400 (regular); 700 (bold) for labels | 1.5 | none; uppercase for labels | Inter |
| Heading (h3/h4) | 20px | 700 (bold) | 1.3 | none | Barlow Condensed |
| Heading (h1/h2) | 32px | 700 (bold) | 1.15 | uppercase | Barlow Condensed |
| Display | 48px | 700 (bold) | 1.1 | uppercase | Barlow Condensed |

Font sizes: 4 (16, 20, 32, 48).
Font weights: 2 (400 regular, 700 bold).

Labels use 16px bold uppercase Inter instead of a separate 14px size. This maintains visual distinction while keeping the type scale tight.

**Responsive scaling for Display:** 36px on mobile (<640px), 48px on tablet+ (>=640px).
**Responsive scaling for h1/h2:** 24px on mobile (<640px), 32px on tablet+ (>=640px).

**Source:** D-06 (condensed sans-serif heading), D-07 (clean sans-serif body), D-08 (ALL CAPS h1/h2 only). Sizes and weights within Claude's discretion.

## Color

### Community Mode (Light — default)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Page background, content surfaces |
| Secondary (30%) | #1B2A4A | Header, footer, nav bar, sidebar backgrounds |
| Accent (10%) | #DC2626 | CTA buttons, active links, highlight bars, urgent banner |
| Destructive | #DC2626 | Same as accent in this brand (red IS the brand) |
| Text primary | #111827 | Body text, headings on light backgrounds |
| Text secondary | #4B5563 | Captions, metadata, supporting text |
| Text on dark | #FFFFFF | Text on navy/dark backgrounds |
| Text on accent | #FFFFFF | Text on red/crimson backgrounds |
| Border | #D1D5DB | Card borders, dividers on light backgrounds |

### Urgent Mode (Dark)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0F172A | Page background (near-black slate) |
| Secondary (30%) | #1E293B | Card backgrounds, elevated surfaces |
| Accent (10%) | #EF4444 | CTA buttons, active links, highlight bars, urgent banner |
| Text primary | #F8FAFC | Body text, headings on dark backgrounds |
| Text secondary | #94A3B8 | Captions, metadata, supporting text |
| Border | #334155 | Card borders, dividers on dark backgrounds |

### Accent Reserved For (both modes)

- Primary CTA buttons (solid red background)
- Active navigation links (text color or underline)
- Inline text links on hover
- Urgent banner background
- Pull quote left border accent
- Card hover top-border highlight
- Logo "UNITED" text

Accent is NEVER used for: backgrounds of content areas, body text, disabled states, decorative borders.

## Prose / Rich Text Contract

The `@tailwindcss/typography` plugin styles CMS-rendered content. Override defaults to match brand:

| Property | Value |
|----------|-------|
| Prose body | 16px Inter, weight 400, line-height 1.75 (wider for long-form) |
| Prose headings | Barlow Condensed 700, h2/h3 uppercase |
| Prose links | Accent color, underline on hover |
| Prose blockquote | Left border 4px accent, italic, slightly larger (18px) |
| Prose images | Full-width, 0px border-radius |
| Prose max-width | 65ch |
