  In src/components/layout/Footer.tsx, add focus-visible ring classes to ALL interactive elements:

  1. The two Button components in the CTA section -- these already have focus ring styles from the Button component itself (`focus:ring-2 focus:ring-accent focus:ring-offset-2`). However, the ring-offset color defaults to white, which creates a visible white gap on the dark navy footer background. Add `className="focus:ring-offset-navy"` prop to both Button elements:
  ```tsx

  <Button href="/meetings" variant="secondary" className="focus:ring-offset-navy">
    Upcoming Meetings
  </Button>
  ```

  2. All four Link elements in the bottom bar navigation. Replace their className to add focus ring classes:
  ```tsx

  ```

  The `ring-offset-navy` ensures the offset gap between ring and element matches the footer's dark navy background instead of defaulting to white. Uses `focus-visible` (not `focus`) so rings only appear on keyboard navigation, not mouse clicks.
