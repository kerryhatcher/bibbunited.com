    For the red contrast verification:
    Calculate the WCAG AA contrast ratio of `#DC2626` on `#FFFFFF` using the relative luminance formula:
    - Relative luminance L = 0.2126*R + 0.7152*G + 0.0722*B (where R,G,B are linearized)
    - Contrast ratio = (L_lighter + 0.05) / (L_darker + 0.05)
    - For `#DC2626`: R=220/255=0.863, G=38/255=0.149, B=38/255=0.149
    - Linearize: R_lin=0.863^2.2~0.725, G_lin=0.149^2.2~0.019, B_lin=0.149^2.2~0.019
    - L_red = 0.2126*0.725 + 0.7152*0.019 + 0.0722*0.019 = 0.154 + 0.014 + 0.001 = 0.169
    - L_white = 1.0
    - Ratio = (1.0 + 0.05) / (0.169 + 0.05) = 1.05 / 0.219 = 4.79:1

    This passes WCAG AA (>4.5:1). If the executor's precise calculation shows it below 4.5:1, change `--color-crimson` in `src/app/(frontend)/styles.css` line 17 from `#DC2626` to `#C62828` and update `--color-accent` on line 32 from `#DC2626` to `#C62828`.
