Per D-03: Sitemap generated via `next-sitemap` package at build time. Per D-04: robots.txt with /admin disallowed.

**1. Install next-sitemap:**
Run `pnpm add next-sitemap` to add the package.

**2. Create `next-sitemap.config.cjs` (NEW):**
```javascript
/** @type {import('next-sitemap').IConfig} */
module.exports = {
  siteUrl: 'https://www.bibbunited.com',
  generateRobotsTxt: true,
  robotsTxtOptions: {
    policies: [
      {
        userAgent: '*',
        allow: '/',
        disallow: '/admin/',
      },
    ],
  },
  // Exclude admin and API routes from sitemap
  exclude: ['/admin/*', '/api/*'],
  // Default change frequency and priority
  changefreq: 'weekly',
  priority: 0.7,
}
```

**3. Add postbuild script to package.json:**
Add `"postbuild": "next-sitemap"` to the `scripts` section of package.json. This runs automatically after `pnpm build` (or `next build`) completes, generating `public/sitemap.xml`, `public/sitemap-0.xml`, and `public/robots.txt` from the built Next.js routes.

Do NOT create `src/app/sitemap.ts` or `src/app/robots.ts` -- next-sitemap handles both at build time per D-03.
